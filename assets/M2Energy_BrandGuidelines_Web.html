04 · Animaciones y Estructura

Animaciones Eléctricas

El sitio incluye un sistema de animaciones CSS/JS que simulan fenómenos eléctricos reales. Son controlables por el usuario vía el panel "Tweaks".

⚡
Arcos Eléctricos
Líneas SVG animadas que simulan arcos de alta tensión. Color ámbar #f6be1f con glow. Intensidad controlable (0–100).
✦
Chispas / Partículas
Partículas pequeñas que se mueven como chispas eléctricas. Canvas JS. Cantidad controlable (0–100).
〰
Flujo de Energía
Ondas sinusoidales animadas que representan corriente eléctrica fluyendo. Velocidad controlable.
🌩
Rayos
Destellos de rayos ocasionales en el hero. Frecuencia controlable. Color ámbar/azul intercalado.
◉
Glow / Resplandor
Efecto de resplandor en arcos y elementos clave. Intensidad controlable (0–20). CSS box-shadow animado.
🎨
Paletas Tweaks
3 paletas: Ámbar + Azul (default), Solo ámbar, Solo azul. Cambian los colores de las animaciones en tiempo real.

05 · Reglas y Stack Técnico

Reglas de Diseño

Guía de qué hacer y qué no hacer al modificar o expandir el sitio web de M2 Energy.

✓ Hacer
  • Usar fondos oscuros #0d1a23 o #0a0a0a para secciones premium
  • Botones CTA siempre en ámbar #f6be1f con texto oscuro
  • Títulos en Barlow Condensed Bold 700
  • KPIs con números grandes en ámbar sobre fondo oscuro
  • Líneas decorativas en ámbar como separadores
  • Cards con border-top de 3px en #f6be1f
  • Mantener las animaciones eléctricas en el hero
  • Usar el panel Tweaks para ajustar animaciones
  • Tags con letter-spacing 3–4px en ámbar
✗ No hacer
  • Usar fondos blancos en secciones hero o premium
  • Cambiar el ámbar por otro tono de amarillo
  • Mezclar Barlow Condensed con fuentes sans genéricas
  • Usar verde #6cb444 como color primario
  • Poner texto oscuro sobre fondos oscuros
  • Usar gradientes de colores ajenos a la paleta
  • Quitar las animaciones — son parte de la identidad
  • Modificar el logo (⚡ entre M2 y Energy)
  • Usar bordes redondeados mayores a 8px en cards

Para actualizar el sitio:
1. Editar en Claude Design claude.ai/design → Exportar HTML → Copiar a ~/m2energy-web/index.html
2. Desde terminal: cd ~/m2energy-web && git add . && git commit -m "descripción" && git push
3. Cloudflare despliega automáticamente en 1–2 minutos.
